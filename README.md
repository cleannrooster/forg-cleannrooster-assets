## cleannrooster — Free Asset Repository

Five years of building a grimdark RPG world in Minecraft, freely available.

This repository contains the accumulated Blockbench models, textures, armor sets, humanoid mobs, weapons, and spell icons produced over this period.
Over 200 spell icons, complete faction mob systems, 
themed armor sets, and ranged weapon progressions.

Everything here is my work, built from nothing over five years.
